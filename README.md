🚀 Latest Student Information System (Serverless AWS Project)
📌 Overview

The Latest Student Information System is a fully serverless cloud-based application built using AWS services.
It allows efficient management of student records with high scalability, low cost, and zero server maintenance.

This project demonstrates real-world serverless architecture, REST API design, and cloud-native development using AWS.

🏗️ Architecture
Client Request
      ↓
API Gateway
      ↓
AWS Lambda (Python Logic)
      ↓
Amazon DynamoDB
      ↓
Response Returned

⚙️ Tech Stack
☁️ AWS Lambda – Serverless backend logic
🗄️ Amazon DynamoDB – NoSQL database
🌐 API Gateway – REST API management
🔐 IAM – Secure access control
📊 CloudWatch – Monitoring & logging
🐍 Python – Backend programming language
🔧 Git & GitHub – Version control
✨ Features
➕ Add student records
🔍 Retrieve student details
✏️ Update student information
❌ Delete student records
⚡ Fully serverless architecture
🔐 Secure API endpoints
📈 Scalable cloud-native design
🔄 Workflow
User sends request via API Gateway
API Gateway triggers AWS Lambda
Lambda executes Python business logic
Data is stored/retrieved from DynamoDB
Response is returned to user
🚀 Deployment Steps
Create DynamoDB table (StudentTable)
Create AWS Lambda function (Python 3.x)
Attach IAM role with required permissions
Configure API Gateway routes (GET, POST, PUT, DELETE)
Deploy API
Test using Postman
