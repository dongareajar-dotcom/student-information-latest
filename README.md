🚀 Latest Student Information System (Serverless AWS Project)










📌 Overview

The Latest Student Information System is a fully serverless cloud application built using AWS services. It is designed to efficiently manage student data with high scalability, low cost, and zero server management.

This project demonstrates real-world cloud architecture, REST API development, and serverless computing.

🏗️ Architecture
Client Request
      ↓
API Gateway
      ↓
AWS Lambda (Python Logic)
      ↓
Amazon DynamoDB
      ↓
Response Returned
⚙️ Tech Stack
☁️ AWS Lambda – Serverless backend logic
🗄️ Amazon DynamoDB – NoSQL database
🌐 API Gateway – REST API management
🔐 IAM – Secure access control
📊 CloudWatch – Monitoring & logging
🐍 Python – Backend programming language
🔧 Git & GitHub – Version control
✨ Features
➕ Add student records
🔍 Retrieve student details
✏️ Update student information
❌ Delete student records
⚡ Fully serverless architecture
🔐 Secure API endpoints
📈 Scalable cloud-native design
🔄 Workflow
User sends request via API Gateway
API Gateway triggers AWS Lambda
Lambda executes Python business logic
Data is stored/retrieved from DynamoDB
Response is returned to user
🚀 Deployment Steps
1. Create DynamoDB table (StudentTable)
2. Create AWS Lambda function (Python 3.x)
3. Attach IAM role with required permissions
4. Configure API Gateway routes (GET, POST, PUT, DELETE)
5. Deploy API
6. Test using Postman
